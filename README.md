# short_note <img src="assets/logo.png" width=20 />

A simple note taking app.

## Features

Take notes, save it, modify it, delete it all are possible.

Each note is consist of

- Title
- Description
- Todos list

### Technology used

- Flutter
- Riverpod
- Sqflite database
